📋 How to Use This Tool

  • Upload the latest Alpha Engineer stock ranking spreadsheet (Excel format)
  • First time use: set your minimum rank threshold for SELL signals
  • First time use: enter your current stock tickers in the portfolio table
  • The tool will automatically show SELL signals for stocks below your minimum rank
  • Sell the stocks and buy new top stocks from the Alpha Engineer stock ranking
  • Optional: use the position sizing guide to determine minimum and maximum buy size of a stock
  • Replace the sold stocks with the newly bought stocks
  • Export your portfolio to save it and import it next time to repeat the process
